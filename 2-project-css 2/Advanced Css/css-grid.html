Grid is a system that helps us to divide the elements (layout) in columns and rows

grid has 12 properties And only 8 are Important and other 4 are optional to Use
    // parent

  3. Col-gap : it is used to define the col gap in grid
  4. Row-gap : it is used to define the row gap in Grid
