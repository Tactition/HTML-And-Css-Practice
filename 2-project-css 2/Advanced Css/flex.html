Flex box

flex box is used when we wanna set the children elements of any container after each other it might be in row and column

there are two axix in the screen or viewport which are as

  • X axis : main Axis --- justify content = it places the items in start center or in the End of main axisis or horizontal Axis from left to right
  • Y axis : cross Axis --- align item = it places the items anyware in the vertical axis so that item or items can be aligned anywere from top to bottom
flex direction : its is the property of flex which orders the items in either row or column by its values like
  1. row
  2. column
  3. row-reverse
  4. column-reverse
Flex wrap : it is used to shrink or unShrink the items of flex container so the item can either shrink and manage thier size according to container or items will not shrink and will keep their original size intact and will shift tho new line if there is no avialibilty of space.
  1. Wrap : boxses will not shrink and will start to adjust themselves in new line
  2. Nowrap : boxes will shrink and will not go to new line
